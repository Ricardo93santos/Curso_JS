O metodo write() é usado para escrever um conteudo na tela. O mesmo deve ser usado com o objeto document.

Ex: document.write(`Olá mundo!`)

Dentro do Write() se colocarmaos "", ''e `` ele interpreta com uma string, onde usando o sinal de + entre duas strings ele se torna uma concatenação.

EX: document.write("Olá"+'mundo'+"!") Será escrito: Olámundo!

Podemos usar tags html e inclusive estilizações de css:
EX:
document.write("Olá mundo"+'!' +" <p style = 'color:red;' >Minha primeira aula de JavaScript<⁄p>")

Ao usarmos o `` podemos colocar variáveis sem precisar concatenar basta usar ${variável}. EX:document.write(`Olá ${variável}!`)
