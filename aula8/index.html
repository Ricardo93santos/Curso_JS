Try, Catch e Finally

Para tratar erros em JavaScript usamos os comando try, catch e finally, onde no bloco de comando try, deve inserir a rotina que queremos trar um provavel erro, essa rotina não gera um erro e sim uma rotina que seja passível de geração de erro.

Após ser identificado o erro no try, o mesmo é passado para o bloco de comando catch, onde será feito o tratamento do erro, que pode ser tratado da maneira que quiser, armazenando ou apresentando para o usuário.

Já o bloco finally vai ser execultado caso ocorra um erro ou não, o mesmo é opcional, podendo ser omitido.

A estrutura do try...catch é a seguinte: 

try{
        Rotina para ser testada
      }catch(e){
        rotina para tratar o erro
      }finally{
        opcional
      }
O comando try testa se tem um erro, identificando o mesmo passa para o comando catch, que armazena o erro na variável que está dentro do parentese, nesse caso "e", e execulta o tratamento dentro do bloco de comando. 

    EX:

    let nome = ""
    try{
      if(nome == ""){
        throw "Preencha o campo nome."
      }
      document.write(nome)
    }catch(e){
      document.write(`${e}`)
    }

Dentro do try será verificado com o comando "if" se a variável "nome" está vazia, o "throw" gera a exceção, com uma infromação e no bloco de comando "catch" é apresentado a informação atravez da variável "e".
