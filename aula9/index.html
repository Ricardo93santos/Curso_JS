Manipulando Elementos do HTML

Existem várias formas de manipular um elemneto no HTML, para isso devemos capturar esse elemento usando os métodos prórios do JavaScript.

getElementById

Como já diz o ome do métoto manipulamos o elemento atrazes do seu id. A sua sitaxe é bem simples, basta usar a propriedade document, seguida do método getElementById, acompanhado com parenteses que conterá o id do elemento.

    EX:
    let nome = document.getElementById("nome")
    nome.style.color="red"

A variável "nome" armazena o elemento do HTML onde pode ser manipulado, alterando a sua cor, tamanho ou sendo usado em alguma função expecífica.

getElementsByTagName

O método “getElementsByTagName” retorna o(s) elemento(s) com o nome indicado, o ponto mais interessante é que caso haja várias tags iguais e isso é bastante comum, o método funciona como um vetor com todas as tags, por exemplo, se houver 10 tags p essas tags podem ser obtidas por array.

    EX:
    let tagP = document.getElementsByTagName("p")[3]
    tagP.style.color="blue"

getElementsByClassName

Funciona semelhante ao getElementsByTagName, mas usado para capturar elementos atravez das suas classes.

    EX:
    let p1 = document.getElementsByClassName("p1")[0]
    p1.style.color="pink"

querySelector

Esse método é usado para capturar o primeiro elemento dentro do documento atrazes de seu id ou de sua class, basta informar entre os parenteses se é um id usando o sinal "#" e se for uma class o "."

    EX:
    let id = document.querySelector("#id")
    let class = document.querySelector(".class")

querySelectorAll

O querySelectorAll() é um método que retorna os elementos do documento que correspondem a uma classe de CSS, como um objeto dentro de uma lista de objetos (array). Uma das grandes vantagens de usar o método querySelectorAll() é que podemos nos referenciar tanto a tags como classes CSS.

  • Obter todos os elementos com a tag p.

    var ps = document.querySelectorAll(“p”)

    ps[0].style.backgroundColor=”#F00″; //Formata a cor de fundo para vermelho do primeiro elemento com a tag.

  • Obter todos os elementos com a tag p que usam a classe aula.

    var ps = document.querySelectorAll(“p.aula”)

    ps[0].style.backgroundColor=”#00F”; //Formata a cor de fundo para azul do primeiro elemento com a tag

    que usa a classe aula.

  • Obter quantos elementos com a tag p que usam a classe aula.

    var qtdeps = document.querySelectorAll(“.aula”).length;

  • Obter todos os elementos que usam a classe aula.

    var ca = document.querySelectorAll(“.aula”);

    var i;

    for (i = 0; i < ca.length; i++) {ca[i].style.backgroundColor=”#0F0″;} //Formatar todos estes elementos a cor de fundo para verde.

  • Obter todos os elementos com a tag que possuem a propriedade “target” configurada

    var as = document.querySelectorAll(“a[target]”);

    var i;

    for (i = 0; i < as.length; i++) {as[i].style.border=”2px solid #000″;} //Formatar a borda de todos estes elementos com largura 2px, linha sólida e cor preto.

  • Obter todos os elementos com a tag

    que estão dentro de uma div

    var pdv = document.querySelectorAll(“div > p”);

    var i;

    for (i = 0; i < pdv.length; i++) {pdv[i].style.backgroundColor=”#FF0″;} ////Formatar a cor de fundo destes elementos com a cor amarelo.

  • Obter todos os elementos com as tags h1, div e span

    var tags = document.querySelectorAll(“h1, div, span”);

    var i;

    for (i = 0; i < tags.length; i++) {tags[i].style.backgroundColor=”#0FF”;} ////Formatar a cor de fundo destes elementos com a cor ciano.
