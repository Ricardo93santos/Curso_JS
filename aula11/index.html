Objeto Math

O objeto Math é uma biblioteca matemática que contem os métodos e constantes para realizar calculos matemáticos. Alguns métodos como seno, coseno, raiz quadrada, gerar número aleatório e constantes como PI, raiz quadrada de 2, raiz quadrada de 1/2.

Constante Descrição
Math.E Retorna o número Euler
Math.PI Retorna o número PI
Math.SQRT2 Retorna a raiz quadrada de 2
Math.SQRT1_2 Retorna a raiz quadrada de 1/2
Math.LN2 Retorna o logaritimo natural de 2
Math.LN10 Retorna o logaritimo natural de 10
Math.LOG2E Retorna base 2 do logaritimo de E
Math.LOG10E Retorna base decimal do logaritimo de E
Método Descrição
abs(x) Retorna o valor absoluto de x
ceil(x) Retorna o valor de x arredondado para cima
floor(x) Retorna o valor de x arredondado para baixo
max(x,y,z,...) Retorna o maior valor dos argumentos
min(x,y,z,...) Retorna o menor valor dos argumentos
pow(x,y) Retorna o valor de x elevado a y
random() Retorna um número aleatório entre 0 e 1
round(x) Retorna o inteiro mais próximo

Para mais métodos e constantes da classe Math, basta acessar a doculmentação no link Math - JavaScript | MDN

Como usar o objeto Math

Para usar uma constate do Math, basta usar uma das prorpiedas citadas na tabela acima. Podemos armazenar em uma variável. EX:

    const PI = Math.PI
    const raizQuadradaDeDois = Math.SQRT2
    const logDecimalEuler = Math.LOG10E

    console.log(PI) // 3.141592653589793
    console.log(raizQuadradaDeDois) // 1.4142135623730951
    console.log(logDecimalEuler) // 0.4342944819032518

Para usarmos os métodos é da mesma maneira, a diferença é que devemos usar um argumento para o método, e podemos armazenar o resultado em uma variável. EX:

    let v = 1.86
    let v1 = 0.9
    let v2 = 2.5

Podemos gerar númoeros aleatórios usando o método random() e arredondalos com o método round(x), como o método random() gera numeros aleatórios de 0 a 1 devemos multiplicar pelo valor máximo que queremos, EX:

    let vl = 61 
    for(i = 0; i<6; i++){
      console.log(Math.round(Math.random()*vl))
    }

Nesse caso será gerado 6 numeros de 0 até 61
