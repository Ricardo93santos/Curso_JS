VAR, LET, CONST

Exitem duas formas de criar variáveis, usando as palavras reservadas VAR e LET. Onde VAR pode ser alterada, redeclarada, sofre Hosting e a mesma possue um escopo global. LET pode ser alterada, não pode ser redeclarada, nem sofre Hosting, mas possue um escopo local, de bloco.

    EX:

    var a = 1;
    var b = 2
    if(a===1){
      var a = 11
      let b = 22
      console.log(a) // 11
      console.log(b) // 22
    }
    console.log(a) // 11
    console.log(b) // 2

Nesse exemplo a variavel "a" sofre alteração tanto no escopo do "if" quanto fora dele pois foi declarada pela palavra reservada VAR, já a variável declarada com let dentro do escopo "if" só sofre alteração no escopo e ao sair ele volta a ser de valor "2", pois seu escopo é local.

A palavra reservada CONST não pode ser alterada, pois é uma constante. 

    EX:
    const a = 1
    a = 10

    Será impresso uma mensagen de erro no console: 
    Uncaught TypeError: Assignment to constant variable.
