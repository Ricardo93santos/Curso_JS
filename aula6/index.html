Array

O objeto Array do JavaScript é um objeto global usado na construção de vetores e matrizes. Arrays são objetos semelhantes a listas que vêm com uma série de métodos embutidos para realizar operações de travessia e mutação. Um array em JavaScript não tem tamanho nem tipos de elementos fixos. Já que o tamanho de um array pode ser alterado a qualquer momento e os dados podem ser armazenados em posições não contíguas.

Um Array pode armazenaz qualquer valor, seja uma string, um objeto, outro array ou até mesmo uma função. A sua sintaxe é a segunte: let arr = [elemento0, elemento1, elemento2, ...], onde os elementos devem ser separados por uma "," para assumir sua posição e devem ser contados e manipulados apartir do indece 0, pois é o primeiro elemento e o último elemento está na posição equivalente ao valor da propriedade length (tamanho).

    Ex:
    let arr = ["vassoura", 2, "pá" ]
    console.log(arr.length) // 3
    console.log(arr[0]) // vassoura
    console.log(arr[1]) // 2
    console.log(arr[arr.length - 1]) // pá

Manipulando Arrays

Já compreendemos que para verificar o comprimento de um array basta usar o metodo array.length, para acessar o primeiro elemento, array[0] e para acessar o ultimo elemento, array[array.length - 1]. Mas para adicionar, remover e procurar um elemento? Devemos usar devemos usar os seguntes métodos:

  • push() - Adicionar um item ao final do Array
  • pop() - Remover um item do final do Array
  • unshift() - Adicionar ao início do Array
  • shift() - Remover do início do Array
  • indexOf() - Procura o índice de um item no Array, se o elemento não existe dentro do array, então retorna o valor -1
  • splice() - Remover um ou mais item pela posição do índice
    EX:
    let frutas = ['Maçã', 'Banana']

    let adicionarFim = frutas.push('Laranja');
    // ['Maçã', 'Banana', 'Laranja']

    let removerFim = frutas.pop();
    // ['Maçã', 'Banana']

    let adicionarInicio = frutas.unshift('Morango')
    // ['Morango', 'Maçã', 'Banana']

    let removerInicio = frutas.shift()
    // ['Maçã', 'Banana']

    let posicao = frutas.indexOf('Maçã')
    // 0
    let posicao = frutas.indexOf('Pera')
    // -1
    
    let removeItem = frutas.splice(posicao, 0)
    // ["Banana"]
    let frutas = ["Pera", "Maçã", "Abacaxi", "Banana", "Laranja"]
    let removerFrutas = frutas.splice(2, 4)
    // ["Abacaxi", "Banana", "Laranja"]

Nesse caso: 

let frutas = ["Pera", "Maçã", "Abacaxi", "Banana", "Laranja"]
