Elemento Date()

A classe Date() retorna a data em um formato completo, com todas as informações nescessárias, dia, mês, ano, minuto, segundo e a diferença do fuso horário de greenwich.

A classe Date() possue métodos para podermos capturar e manipular elementos da mesma. Por exemplo, para obter o dia do mês usar o método, getDate e para incrementar alguns dias, setDate

Para trabalhar com a classe Date(), devemos criar um objeto do tipo Date, "new Date()", podendo armazenar o mesmo em uma variável, "let data = new Date()".

    EX:
    let data = new Date()
    document.write(data) //Mon Jun 27 2022 11:29:44 GMT-0300 (Horário Padrão de Brasília)

Metodos mais usados do objeto Date()

Método Descrição Exemplo
getDay() Retorna o dia da semana, iniciando com Domingo = 0 let diaSemana = data.getDay()
getDate() Retona dia do mês let dia = data.getDate()
getMonth() Retona o mês, iniciando com Janeiro = 0 let mes = data.getMonth()
getFullYear Retona o ano com 4 digitos, 2022 let ano = data.getFullYear()
getHours() Retona a hora let dia = data.getHours()
getMinutes() Retona os minutos let dia = data.getMinutes()
getSeconds() Retona os segundos let dia = data.getSeconds()
toDateString() Retorna a data por estêncil, padrão americano let dataTexto = data.toDateString()
toLocaleDateString() Retona a data no formato numérico, dd/mm/aaaa let dataNumero = data.toLocaleDateString()
toLocaleString() Retona a data e as horas, dd/mm/aaaa hh:mm:ss let dataEHoraCompleta = data.toLocaleString()
