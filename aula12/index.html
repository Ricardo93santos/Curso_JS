Funcões

Uma função é um conjunto de instruções que executa uma tarefa ou calcula um valor. Para usar uma função, você deve defini-la em algum lugar no escopo do qual você quiser chamá-la.

Para declarar uma função devemos usar a palavra reservada function, seguida pelo nome da função, argumentos para a função dentro de () e declaração que a função vai execultar dentro de {}

    let valor = 2
    let valor1 = 5
    function multiplicar(valor, valor1){
      return valor*valor1
    }
    A declaração return especifica o valor retornado pela função.

Para a função ser execultada a mesma deve ser chamada, onde podemos escrever o nome da função seguida de parenteses (). EX: 

    let valor = 2
    let valor1 = 5
    function multiplicar(valor, valor1){
      return valor*valor1
    }
    console.log(multiplicar(valor, valor1)) //10
    console.log(multiplicar(valor, valor1)) //10
    console.log(multiplicar(valor, valor1)) //10
    A função será chamada 3 vezes, onde o resultado será escrito no console.
