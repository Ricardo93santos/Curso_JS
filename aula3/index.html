Alert, Prompt, Confirm

Caixas de mensgens são usadas para informar ou coletar alguma informação do usuário. Diferente do metodo document.write(), a caixa de mensagem é manipulada pela window e não pelo document. Onde não tem a necessidade de utilizar o elemento window, basta colocar o comando direto.

Alert

A caixa de mensagem Alert possue uma String e um botão de OK, usada para alertar ou informar o usuário de algo.

    EX:
    alert("Olá Mundo!")

Podemos trabalhar com qualquer valor na caixa de mensagem, variaveis, strings, números, calculos.

    EX:
    let som = 2+5
    alert(`A soma das variáveis é ${som}`) // Asoma das variáveis é 7
    alert(2+5) // 7
    alert(`Podemos calcular como: som-1 = ${som-1}`) // Podemos calcular como: som-1 = 6

OBS:

Para quebrar linhas dentro de uma caixa de mensagem devemos usar \n, pois o <br> não vai funcionar.

    EX:
    alert("Olá Mundo! \n Usando caixa de mensagem!")
  
    Vai aparecer:
    Olá Mundo!
    Usando caixa de mensagem!

Prompt

A caixa de mesnsagem Prompt possue uma área de texto, dois botões um de OK e outro de cancelar. Usada para receber alguma informação e armazena-la dentro de uma variável. Todo valor é recebido como uma string.

    EX:
    let idade = prompt(`Informe sua idade:`)

Confirm

O Confirm vai exibi um texto, dois botãoes um de OK e outro de Cancelar. Ele retorna um valor booleano, onde ao clicar em OK retorna true e se clicar em Cancelar retorna false.

    EX:
    let confirmacao = confirm(`Você quer permanecer na página?`)
    if(confirmacao == true){
      document.witre(`Pode acessar`)
    }else{
      document.write(`Não pode acessar`)
    }
